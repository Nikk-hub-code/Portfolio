# 🖥️ Terminal Portfolio – Kaushal Kumar Jha

A futuristic **terminal-style personal portfolio website** with an interactive robot assistant, command-based navigation, and speech synthesis.  
Inspired by developer terminals and AI interfaces.

---

## 🚀 Live Preview
👉 *(https://nikkode.vercel.app/)*

---

## ✨ Features

- 🧑‍💻 **Terminal-based UI** (command-driven interface)
- 🤖 **Animated Robot Assistant**
  - Eye tracking
  - Thinking / success / error states
- 🔊 **Text-to-Speech responses** using Web Speech API
- ⌨️ **Command History Navigation**
  - Arrow Up / Arrow Down support
- 🎨 **Retro cyberpunk styling**
  - Glowing neon effects
  - Scanline overlay
- 📱 **Responsive layout**
- ⚡ Smooth animations and typing effects

---

## 🧾 Available Commands

| Command     | Description                          |
|------------|--------------------------------------|
| `help`     | List all available commands           |
| `whoami`   | About me                              |
| `skills`   | Technical skills                      |
| `projects` | Projects overview                    |
| `education`| Educational background               |
| `contact`  | Contact details                      |
| `clear`    | Clear the terminal                   |

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3** (Flexbox, animations, glow effects)
- **Vanilla JavaScript**
- **Web Speech API**
